Sell rules
  • Hit planned stop → exit, do not negotiate.
  • MBI falls into avoid zone → reduce aggressively.
  • 52W lows expand hard → protect capital first.
  • At +5% to +7% → partial book and trail rest.
  • Past max hold window → review for time stop.
Sizing anchors
  • Calculator first, conviction second.
  • Risk in rupees matters more than gut feel.
  • Reduce size automatically in cautious zones.
  • Never let pyramiding break planned-risk discipline.
  • Use the journal to review which size regimes work best.

  3. Enable the Google Drive API only if you want Drive backup.
  4. Deploy this folder with Firebase Hosting free tier.
